# Contributing to DrCG Viz Chart

Thank you for your interest in contributing! This project is intended as an open, collaborative chart generation tool for PhD thesis research.

## ✅ How to Contribute

### 1) Open an Issue
If you find a bug or have a feature request, open an issue and explain:
- What you tried
- What you expected
- What actually happened

### 2) Send a Pull Request
1. Fork the repository
2. Create a branch: `feature/my-cool-feature`
3. Make your changes
4. Add tests or verify existing tests pass
5. Submit a pull request

### 3) Code Style
- Use consistent naming and formatting (PEP8 for Python)
- Keep UI changes minimal unless they improve usability
- Add comments and documentation for non-obvious logic
